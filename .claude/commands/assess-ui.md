# Assess UI Layer

Read the agent definition at `.claude/agents/06-ui-layer.md` and execute a thorough assessment of the frontend.

## Instructions
1. Read `.claude/agents/06-ui-layer.md` for the full checklist
2. Scan all pages in `app/**/page.tsx` and layouts in `app/**/layout.tsx`
3. Scan all components in `components/`
4. Check `public/` for PWA assets (manifest, service worker)
5. Review Tailwind config and styling consistency
6. Evaluate component architecture, accessibility, and UX patterns
7. Write your findings to `.claude/reports/ui-report.md`
8. Return a brief summary of UX and accessibility gaps

Focus on: component completeness, accessibility, responsive design, and user experience.
