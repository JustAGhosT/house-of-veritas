# House of Veritas - Document Templates

This file lists all 19 governance documents that need to be created as templates in DocuSeal.

## Template Creation Guide

For each document:

1. Create the document content in your preferred editor (Word, Google Docs)
2. Export as PDF
3. Upload to DocuSeal
4. Add signature fields at appropriate locations
5. Configure signing workflow

---
